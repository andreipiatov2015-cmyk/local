# Astra Monitor - Панель управления сайтом

Система **безопасного управления** сайтом для **Astra Linux 1.8**

## ВАЖНО: Новая архитектура

### Что НЕ делает приложение:

- Не устанавливает системные пакеты (nginx, ffmpeg, chromium)
- Не использует `apt install`
- Не загружает пакеты из интернета
- Не изменяет существующую конфигурацию
- Не перезапускает nginx, ffmpeg, chromium

### Что ДЕЛАЕТ приложение:

- Только мониторинг и управление
- Безопасное обновление через временную директорию
- Сохранение всех существующих настроек

## Возможности

### Управление сервисами
- Start/Stop/Restart для всех сервисов одним кликом
- Статус всех сервисов в реальном времени
- PID, порты, uptime каждого процесса

### Системный мониторинг
- Загрузка CPU (общая и по ядрам)
- Использование оперативной памяти
- Использование дискового пространства
- Load Average
- Список открытых портов
- Топ процессов по CPU и памяти

### GitHub синхронизация
- Подключение к GitHub по SSH
- Безопасное обновление через временную папку

## Установка

```bash
# Установка панели (без системных пакетов)
cd astra-monitor
chmod +x install.sh
sudo bash install.sh

# Запуск
astra-monitor
```

## Безопасное обновление

Алгоритм обновления (НИКОГДА не делает git pull в рабочей папке):

1. Создать временную директорию `/tmp/astra-monitor-update`
2. Клонировать новую версию туда
3. Сравнить версии
4. Скопировать только файлы приложения
5. НЕ выполнять install.sh
6. НЕ трогать nginx, ffmpeg, chromium
7. Перезапустить только сервисы приложения
8. Удалить временную директорию

## Безопасность

- Приложение НЕ изменяет системные компоненты
- Обновление происходит через отдельную временную директорию
- Резервное копирование перед каждым обновлением
- Возможность отката к предыдущей версии
