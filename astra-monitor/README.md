# Astra Monitor

Система мониторинга и управления сайтом для **Astra Linux 1.8**

## Возможности

### 💻 Системный мониторинг
- Загрузка CPU (общая и по ядрам)
- Использование оперативной памяти
- Использование дискового пространства
- Load Average
- Список открытых портов
- Топ процессов по CPU и памяти
- Сетевая статистика

### 🌐 Мониторинг сайта
- Статус всех компонентов сайта (Python приложения, Nginx, RTMP)
- Проверка HTTP сервисов
- Количество записей и пресетов
- Размер базы данных и логов
- Активные и неактивные компоненты

### 🔄 Управление обновлениями
- Проверка обновлений из git-репозитория
- Pull изменений
- Автоматический backup перед обновлением
- Перезапуск сервисов после обновления
- История коммитов

## Установка

### Автоматическая установка (рекомендуется)

```bash
cd /путь/к/astra-monitor
sudo python3 setup.py install
```

### Ручная установка зависимостей

```bash
sudo apt update
sudo apt install python3-pip python3-pyqt5 python3-psutil python3-gi
pip3 install --user psutil requests
```

### Запуск

```bash
astra-monitor
```

Или через меню приложений Astra Linux.

## Структура проекта

```
astra-monitor/
├── src/
│   ├── core/           # Ядро мониторинга
│   │   ├── system_monitor.py   # Системный мониторинг
│   │   ├── site_monitor.py     # Мониторинг сайта
│   │   └── git_updater.py      # Обновление из git
│   ├── gui/            # Графический интерфейс
│   │   └── main_window.py      # Главное окно
│   └── utils/          # Утилиты

├── astra_monitor.py    # Точка входа
└── setup.py            # Скрипт установки
```

## Требования

- Astra Linux 1.8 (Smolensk/SE 1.8)
- Python 3.8+
- PyQt5
- psutil
- requests
- Root-доступ (для полного мониторинга)

## Безопасность

⚠️ Приложение требует root-доступа для:
- Чтения информации о всех процессах
- Доступа к портам и сетевой статистике
- Перезапуска системных сервисов
- Обновления файлов сайта
