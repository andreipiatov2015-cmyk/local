# Astra Monitor - Панель управления сайтом

Система **автоматической установки и управления** сайтом для **Astra Linux 1.8**

## 🎯 Возможности

### 🚀 Автоматическая установка
- **Мастер установки** - пошаговый wizard для новичков
- Автоматическая установка всех зависимостей (Nginx, Python, ffmpeg)
- Настройка Nginx с proxy на Flask приложения
- Установка Nginx RTMP для потокового видео
- Создание systemd сервисов для автозапуска
- Копирование файлов сайта

### ⚙️ Управление сервисами
- **Start/Stop/Restart** для всех сервисов
- Статус всех сервисов в реальном времени
- PID, порты, uptime процессов
- Автоматический мониторинг (обновление каждые 3 сек)

### 💻 Системный мониторинг
- Загрузка CPU (общая и по ядрам)
- Использование оперативной памяти
- Использование дискового пространства
- Load Average
- Список открытых портов
- Топ процессов по CPU и памяти
- Сетевая статистика

### 🌐 Мониторинг сайта
- Статус всех компонентов сайта (Flask, Nginx, RTMP)
- Проверка HTTP сервисов (порты 8083, 8084, 8080, 8082)
- Количество записей и пресетов
- Размер базы данных и логов
- Активные и неактивные компоненты

### 🔄 Управление обновлениями
- Проверка обновлений из git-репозитория
- Pull изменений с автоматическим backup
- Автоматический перезапуск сервисов
- История коммитов

## 📋 Сервисы под управлением

| Сервис | Порт | Описание |
|--------|------|---------|
| Nginx | 80 | Веб-сервер |
| Nginx RTMP | 1935 | RTMP стриминг |
| Live Server | 8083 | Основное Flask приложение |
| Reboot Server | 8084 | Управление трансляциями |
| VK Pusher | - | Ретрансляция в VK |
| Xvfb + Chromium | - | Яндекс авторизация |
| x11vnc | 5901 | VNC сервер |
| websockify | 6080 | noVNC прокси |

## 🚀 Установка

```bash
cd /путь/к/astra-monitor

# Вариант 1: Bash скрипт (рекомендуется)
chmod +x install.sh
sudo bash install.sh

# Вариант 2: Python setup
chmod +x setup.py
sudo python3 setup.py

# Вариант 3: Makefile
sudo make install
```

### Зависимости

- Astra Linux 1.8 (Smolensk/SE) или Ubuntu/Debian
- Python 3.8+
- PyQt5
- psutil
- Root-доступ

## 🔒 Безопасность

⚠️ Приложение требует root-доступа для:
- Установки системных пакетов
- Настройки Nginx и systemd
- Запуска/остановки сервисов
- Доступа к портам и системной статистике

## 📝 Использование

### Запуск
```bash
astra-monitor
```

Или через меню: **Приложения → Система → Astra Monitor**

### Мастер установки
Через меню: **Файл → Мастер установки...**

### Управление сервисами
1. Откройте вкладку **"⚙️ Сервисы"**
2. Нажмите нужную кнопку (Запустить/Остановить/Перезапустить)
3. Статус обновляется автоматически каждые 3 секунды
